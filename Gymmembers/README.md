# 🏋️ Gym Members Exercise Dataset

## 📖 Descripción

Este conjunto de datos ofrece una visión integral sobre las rutinas de entrenamiento, las características físicas y las métricas de rendimiento de **973 miembros de un gimnasio**.

Cada registro combina información demográfica, medidas corporales y variables relacionadas con el ejercicio, permitiendo analizar el comportamiento físico de los usuarios, su evolución y la relación entre distintos factores que influyen en su condición física y salud.

---

## 📊 Información del conjunto de datos

- **Número de registros:** 973
- **Tipo de datos:** Variables demográficas, fisiológicas y de entrenamiento.
- **Aplicación:** Análisis estadístico, ciencia de datos, aprendizaje automático e investigación en salud y actividad física.

---

# 📋 Variables del dataset

| Variable | Descripción |
|-----------|-------------|
| **Age** | Edad del miembro del gimnasio. |
| **Gender** | Género del miembro (*Male* o *Female*). |
| **Weight (kg)** | Peso corporal en kilogramos. |
| **Height (m)** | Estatura en metros. |
| **Max_BPM** | Frecuencia cardíaca máxima durante el entrenamiento. |
| **Avg_BPM** | Frecuencia cardíaca promedio durante la sesión. |
| **Resting_BPM** | Frecuencia cardíaca en reposo antes del entrenamiento. |
| **Session_Duration (hours)** | Duración de la sesión de entrenamiento (horas). |
| **Calories_Burned** | Calorías quemadas durante la sesión. |
| **Workout_Type** | Tipo de entrenamiento realizado (*Cardio*, *Strength*, *Yoga* o *HIIT*). |
| **Fat_Percentage** | Porcentaje de grasa corporal. |
| **Water_Intake (liters)** | Consumo de agua durante los entrenamientos (litros). |
| **Workout_Frequency (days/week)** | Número de entrenamientos por semana. |
| **Experience_Level** | Nivel de experiencia (1 = Principiante, 2 = Intermedio, 3 = Experto). |
| **BMI** | Índice de Masa Corporal (IMC). |

---

# 🎯 Objetivos de análisis

Este conjunto de datos permite responder preguntas como:

- 📈 ¿Qué factores influyen en la cantidad de calorías quemadas?
- ❤️ ¿Existe relación entre la frecuencia cardíaca y la intensidad del entrenamiento?
- 🏃 ¿Cómo afecta la experiencia del usuario a su rendimiento físico?
- 💧 ¿El consumo de agua influye en el desempeño durante el ejercicio?
- ⚖️ ¿Qué relación existe entre el IMC, el porcentaje de grasa corporal y el tipo de entrenamiento?
